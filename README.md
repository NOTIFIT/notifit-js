# NOTIFIT Javascript SDK

![NOTIFIT](http://i.imgur.com/uS9hya3.png?1 "NOTIFIT")

This Javascript package creates layer to communicate with NOTIFIT back-end.

## How to use it

### Install it

```bash
npm install notifit
```

### You can test it
```bash
cd node_modules/notifit
npm test
```

### Require it

```javascript
var sdk = require('notifit')
```
